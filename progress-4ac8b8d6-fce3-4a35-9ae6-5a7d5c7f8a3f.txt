## Codebase Patterns

- **cli/ is a standalone Go module** with its own go.mod, separate from api/ and client/
- **Compression utilities** live in `cli/util/util.go` — CompressFiles, GetAllAllowedFilesAndDirsFromPath, GetZipFriendlySize, DeleteHuskyFile
- **Extension filtering** uses an explicit allowlist approach: image/video extensions are blocked, everything else is allowed
- **ZIP destination** is always `~/.huskyci/compressed-code.zip` via `config.GetHuskyZipFilePath()`
- **Error types** are defined in `cli/errorcli/` package (e.g., `ErrInvalidExtension`)
- **Tests in cli/util/** use standard `testing` with `t.Run` subtests and `t.Parallel()` (not Ginkgo/Gomega). `t.TempDir()` provides automatic cleanup.
- **CompressFiles depends on `config.GetHuskyZipFilePath()`** which calls `os.UserHomeDir()` — test isolation requires overriding `$HOME` env var.
- **CompressFiles resolves archive-internal paths relative to CWD** — the new archive/zip implementation uses `filepath.Abs(name)` + `filepath.Rel(cwd, absName)` to compute relative paths safely. Tests `os.Chdir` to the directory containing source files so archive paths are correct.
- **filepath.Rel requires both paths to be absolute or both relative** — always use `filepath.Abs` to resolve relative inputs before calling `filepath.Rel` with an absolute base.
- **filepath.Walk with absolute root produces absolute path arguments** — safe for `filepath.Rel(cwd, path)`.
